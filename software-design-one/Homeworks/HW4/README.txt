To compile and run Virtual Pet you will need to follow these  steps: 

open git bash and navigate to where the VirtualPet.java file is located 
enter javac VirtualPet.java into git bash
enter java VirtualPet into git bash 
the program should then run and you should be able to play the game 

Known bug issues:

several days into the game it will not display the pet status on certain days
the game has done this as early as day seven and as late as day twenty one  
so far hunger level and cleanliness level have been the only status effected
